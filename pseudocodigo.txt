SISTEMA PARA RECEITA OFTALMICA 


VAR 
GRAUEE, GRAUED, GRAUCE, GRAUCD = INTEIRO

INICIO PROGRAMA
ESCREVA ("DIGITE O GRAU ESFERICO DO OLHO ESQUERDO")
LEIA (GRAUEE)
SE GRAUEE FOR > -15
ESCREVA ("Dado inválido, revise os dados e tente novamente")
ELSE


ESCREVA ( "GRAU ESFERICO OLHO DIREITO" )
LEIA (GRAUED)
SE (GRAUED) FOR > -15
ESCREVA ("Dado inválido, revise os dados e tente novamente")
ELSE


ESCREVA ("GRAU CILINDRICO OLHO ESQUERDO")
LEIA (GRAUCE)
SE (GRAUCE) FOR > -6
ESCREVA ("Dado inválido, revise os dados e tente novamente")
ELSE


ESCREVA ("GRAU CILINDRICO OLHO DIREITO")
LEIA (GRAUCD)
SE (GRAUCD) FOR > -6
ESCREVA ("Dado inválido, revise os dados e tente novamente")
ELSE

LEIA (GRAUEE + GRAUED)
ESCREVA ("SEU GRAU ESFERICO É DE (GRAUEE + GRAUED)")

LEIA (GRAUCE + GRAUCD)
ESCREVA ("SEU GRAU CILINDRICO É DE (GRAUCE + GRAUCD")


SE (GRAUEE + GRAUED = ENTRE -3 e -12 E GRAUCE + GRAUCD <= -2
ESCREVA ("A LENTE QUE ATENDE SUA RECEITA É A LENTE PRIME")
ELSE 
ESCREVA ("A LENTE QUE ATENDE SUA RECEITA É A LENTE VISION")

ENQUANTO < QUANTIDADE LENTE PRIME ESTOQUE >0 > FAÇA
FIM ENQUANTO

FIM PROGRAMA

O SISTEMA PRECISA EXIBIR UMA OPÇÃO DE LENTE QUE ATENDA A RECEITA INFORMADA

PRECISA COLETAR 4 DADOS NUMERICOS
GRAU ESFERICO DO OLHO ESQUERDO
GRAU ESFERICO OLHO DIREITO
GRAU CILINDRICO OLHO ESQUERDO
GRAU CILINDRICO OLHO DIREITO

LENTE PRIME = GRAU ESFERICO OLHO ESQUERDO OU DIREITO ENTRE -3 E -12
E GRAU CILINDRICO ATÉ -2 

SE GRAU CILINDRICO = GRAU ESFERICO ENTRE -3 E -10


LENTE VISION 0 A -15 DE ESFERICO E ATÉ -5 DE CILINDRICO
SE LENTE PRIME DISPONIVEL SER A UNICA A APARECER

GRAU LIMITE É DE 0 A -15 PARA ESFERICO E -6 PARA CILINDRICO E
